Encode is the CS club at PDEU, and I led the web dev core committee team for the 2024 edition. The site has this hacker-themed aesthetic - dark green on black, styled like a terminal window with a command prompt. It handles event management, member profiles, and resource sharing for the club.

Built with the MERN stack and TailwindCSS. Worked on the homepage alongside Nemil Shah. The terminal design was a deliberate choice - it's a CS club, so the site should feel like something its members would actually appreciate. The backend handles event CRUD, member data, and gallery management.
