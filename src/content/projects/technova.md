Built for the Pre TechNova Hackathon with Dhruv Maradiya. We tackled four transportation problems in one system - GRU-based traffic congestion prediction, ResNet-34 driver fatigue and distraction detection, CNN-based accident detection from surveillance footage, and dynamic bus schedule optimization.

The backend is FastAPI with ML models in PyTorch and TensorFlow. The frontend is Next.js with TypeScript. We deployed the ML services on AWS and Render, and the frontend on Vercel. It was an ambitious scope for a hackathon but we got all four modules working end to end.
