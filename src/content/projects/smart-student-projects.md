This was actually my first-ever website project, built for Smart India Hackathon 2023. The idea was an online platform that compiles and showcases student projects from universities and colleges across India - promoting knowledge sharing and collaboration between students and educators.

It has project submission, discovery, user profiles, and a ranking leaderboard. Built with plain HTML, CSS, and JavaScript with a Firebase backend. Looking back at the code is humbling, but it was the project that got me into web development, so it holds a special place.
