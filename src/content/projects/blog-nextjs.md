My old blog platform before I rebuilt it in Astro. It had static and incremental static generation, full Markdown support, a comments system, social sharing, search, SEO optimization, analytics, dark and light mode, auto-generated table of contents, and responsive design throughout.

Built with Next.js, TypeScript, TailwindCSS, and SQLite for the comments backend. Deployed on Vercel. I eventually moved to Astro because I wanted something lighter and more content-focused, but this was a solid platform that taught me a lot about static site generation and content management.
