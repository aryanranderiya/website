A platform where you describe browser tasks in plain English and it executes them. The FastAPI backend handles session management, captcha detection, and content extraction while a Next.js frontend gives you real-time progress updates and task tracking.

It supports intelligent command chaining, so you can describe multi-step workflows and it figures out the sequence. Content extraction covers text, tables, links, and JSON-LD structured data. Built this as an exploration into what practical browser automation looks like when you remove the scripting overhead entirely.
