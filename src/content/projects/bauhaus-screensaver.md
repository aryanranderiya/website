I really liked the Bauhaus Clock screensaver but it was a paid app. So I just built an open-source version of it. It's a macOS .saver file written in Objective-C - a minimal geometric clock inspired by the Bauhaus design movement. You download the compiled file from the releases page and double-click to install.

It was mostly an excuse to play with Objective-C and macOS screensaver APIs, and to scratch an itch for having a beautiful clock on my screen without paying for it.
