Built for my Operating Systems class at PDEU. It's a web app that visualizes three classic OS algorithms - Optimal Page Replacement, the Producer-Consumer Problem, and Shortest Seek Time First disk scheduling.

The backend is Flask and the frontend is plain HTML, CSS, and JavaScript. Each algorithm has an interactive visualization so you can step through the execution and understand what's happening at each stage. Deployed on Vercel. It was a nice way to make abstract OS concepts tangible.
