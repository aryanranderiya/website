Built for a Frontend UI Hackathon hosted by Outlier.ai. It's a pixel art adventure game where you explore dungeons and collect items - classic RPG vibes with swords, shields, knight helmets, and torch-lit corridors.

Everything runs in the browser with Next.js and TypeScript. No game engine, just React components and CSS doing the heavy lifting. It was a fun exercise in seeing how far you can push a web framework for something that doesn't feel like a typical web app.
